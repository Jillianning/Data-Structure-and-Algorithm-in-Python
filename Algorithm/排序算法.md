# 基础排序算法总结

# 时间复杂度及运行时间
* 三种排序算法时间复杂度都是O(nlogn)
* 一般情况下，运行时间而言：`快速排序 < 归并排序 < 堆排序`

# 缺点
* 快速排序：极端情况下排序效率低
* 归并排序：需要额外的内存开销
* 堆排序：在三种排序算法中相对较慢

# 基础排序算法对比
| 排序算法   | 时间复杂度（最坏、平均、最好情况 ）     | 空间复杂度   | 稳定性 |  代码复杂度 |
| ----      |  ---------------------------      | --------   | -----   | ---------
| 冒泡排序   | O(n^2)， O(n^2)， O(n^2)           | O(1)       | 稳定   |  简单      |  
| 选择排序   | O(n^2)， O(n^2)， O(n^2)           | O(1)       | 不稳定  | 简单      |
